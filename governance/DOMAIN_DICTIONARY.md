## المصطلحات المالية والمحاسبية

| المصطلح (عربي) | المصطلح (إنجليزي) | التعريف الدقيق |
|---|---|---|
| قيد محاسبي | Journal Entry | سجل مالي متوازن (مدين = دائن). لا يُعدَّل بعد الترحيل — يُعكس فقط. |
| ترحيل | Posting | تحويل قيد من Draft → Posted. عملية لا رجعة فيها. |
| فترة محاسبية | Accounting Period | شهر محاسبي. بعد الإقفال لا كتابة. |
| إقفال الفترة | Period Close | 7 خطوات. بعد Hard Close لا فتح. |
| شجرة الحسابات | Chart of Accounts | النظام المحاسبي الموحد العراقي. |
| مدين | Debit | الجانب الأيسر من القيد. |
| دائن | Credit | الجانب الأيمن من القيد. |
| ذمم مدينة | Accounts Receivable (AR) | ما يجب على العملاء دفعه. |
| ذمم دائنة | Accounts Payable (AP) | ما يجب علينا دفعه للموردين. |
| مطابقة بنكية | Bank Reconciliation | مقارنة كشف البنك بسجلات الدفاتر. |
| Hash Chain | Hash Chain | SHA-256 متسلسل على كل قيد — يكتشف أي تلاعب. |
| Double-Entry | Double-Entry | كل معاملة مالية لها مدين ودائن متساويان. قانون لا يُكسر. |

## المصطلحات التشغيلية

| المصطلح | الإنجليزي | التعريف |
|---|---|---|
| فاتورة مبيعات | Sales Invoice | مستند رسمي للبيع. بعد Submit → Immutable. |
| أمر شراء | Purchase Order (PO) | طلب رسمي للشراء من مورد. |
| إيصال استلام | Goods Receipt Note (GRN) | توثيق استلام البضاعة فعلياً. |
| مطابقة ثلاثية | 3-Way Match | PO ↔ GRN ↔ Invoice. لا دفع بدون تطابق. |
| مرتجع | Return | إعادة سلعة. دائماً بقيد عكسي — لا حذف. |
| وردية | Shift | فترة عمل الكاشير. لها افتتاح وإغلاق وتسويات. |
| تقرير X | X-Report | ملخص وردية قيد التشغيل — يتكرر. |
| تقرير Z | Z-Report | ملخص وردية نهائي — مرة واحدة فقط. |
| نقطة إعادة الطلب | Reorder Point | الكمية التي عندها يُنشأ طلب شراء تلقائي. |
| متوسط متحرك موزون | Moving Weighted Average (MWA) | طريقة تقييم المخزون المعتمدة. |
| StockLedger | Stock Ledger | سجل حركة المخزون. Append-Only — لا حذف أبداً. |

## مصطلحات المخزون والمنتجات

| المصطلح | الإنجليزي | التعريف |
|---|---|---|
| قالب المنتج | Product Template | الأب. يحتوي خصائص مشتركة لكل المتغيرات. |
| متغير المنتج | Product Variant | الابن. له SKU وباركود ومخزون خاص به. |
| وحدة القياس | Unit of Measure (UoM) | الوحدة الأساسية لتتبع المخزون (حبة، كيلو...). |
| رصيد متاح | Available Qty | = qty_on_hand - qty_reserved. هذا ما يُباع. |
| رصيد محجوز | Reserved Qty | محجوز لطلبات معلقة — لا يُباع. |
| كمية واردة | Incoming Qty | في PO مؤكد، لم تصل بعد. |
| جرد دوري | Cycle Count | جرد 10-20% من المخزون أسبوعياً بدون توقف. |
| جرد شامل | Physical Inventory | جرد كامل للمستودع — يوقف الحركة مؤقتاً. |
| فرق الجرد | Stock Variance | countedQty - systemQty. يُرحَّل محاسبياً. |

|---|---|
| RBAC | Role-Based Access Control — ما تستطيع فعله |

## مصطلحات الذكاء الاصطناعي

| المصطلح | التعريف |
|---|---|
| Tier 1 | Qwen 7B — lazy loaded، للقرارات المعقدة |
| Tier 2 | PyOD + Prophet + scikit-learn — دائم التشغيل، خفيف |
| Tier 3 | قواعد برمجية ثابتة — فوري، لا يخطئ، لا يحتاج AI |
| Lazy Loading | تحميل Qwen عند الطلب وتفريغه بعد دقيقتين من الخمول |
| Anomaly | معاملة تخرج عن النمط الطبيعي — Tier 2 يكتشفها |
| NLQ | Natural Language Query — سؤال عربي ← SQL آمن ← نتيجة |

## اختصارات يُحظر فيها اللبس

| الاختصار | المقصود | لا يعني |
|---|---|---|
| SO | Sales Order | لا يعني "Shift Open" |
| PO | Purchase Order | لا يعني "Point of..." |
| GRN | Goods Receipt Note | لا يعني "Green" أو غيره |
| MWA | Moving Weighted Average | لا يعني FIFO أو LIFO |
| COD | Cash on Delivery | لا يعني "Code" |
| CJO | Custom Job Order | طلب منتج مخصص |
| RLS | Row Level Security | PostgreSQL feature |
| JE | Journal Entry | القيد المحاسبي |
