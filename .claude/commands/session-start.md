بداية جلسة جديدة — اقرأ حالة المشروع واستعد للعمل.

نفّذ بالترتيب بالضبط:

1. اقرأ `governance/SESSION_HANDOFF.md`
2. اقرأ `governance/MODULE_STATUS_BOARD.md`
3. اقرأ `governance/OPEN_ISSUES.md`
4. اقرأ `governance/DECISIONS_LOG.md` (آخر 10 قرارات)
5. راجع آخر "الخطوة التالية" من SESSION_HANDOFF

```

║  📍 حالة المشروع — جلسة جديدة              ║
╠══════════════════════════════════════════════╣
║  المرحلة الحالية: [من MODULE_STATUS_BOARD]  ║
║  Wave: [X] — الموجة الحالية                 ║
║  آخر إنجاز: [من SESSION_HANDOFF]            ║
║  الخطوة التالية: [من SESSION_HANDOFF]       ║

╚══════════════════════════════════════════════╝
```
